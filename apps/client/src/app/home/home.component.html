Código aberto

O projeto WebRTC é de código aberto e suportado pela Apple, Google, Microsoft e Mozilla, entre outros.

Esta página será mantida pela comunidade.

Versátil

Projetado de forma que possa ser integrado aos sistemas de comunicação existentes: voz sobre IP (VOIP), vários clientes SIP.

Até mesmo a rede telefônica pública comutada (PSTN).

Simples

Aplicativos RTC de áudio, vídeo e troca de dados ponto a ponto com alta qualidade, requer muitas novas funcionalidades no navegador.

Felizmente, o navegador abstrai a essa complexidade com três APIs principais.
